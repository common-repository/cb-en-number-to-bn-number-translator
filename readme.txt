5. Tags

6. Date

7. View

etc....


== Installation ==

* Just Upload `CB En Number to BN Number Translator` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* And enjoy this plugin.
